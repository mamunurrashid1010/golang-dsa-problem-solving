# Go (Golang) DSA Problem Solving

My personal practice repository for Data Structures and Algorithms (DSA) and problem-solving using Go (Golang). I continuously add and solve new problems to improve my algorithmic thinking and Go programming skills.


## Problems

| # | Problem | Description |
|---|---------|-------------|
| 1 | [Even/Odd Number Check](./1.even_odd_number_check) | Check if a given number is even or odd |
| 2 | [Positive/Negative Number Check](./2.positive_negative_number_check) | Determine if a number is positive, negative, or zero |
| 3 | [Sum of 1 to N](./3.one_to_n_number_sum) | Calculate the sum of numbers from 1 to N |
| 4 | [Even Number Print](./4.even_number_print) | Print all even numbers in a given range |
| 5 | [Factorial](./5.factorial) | Calculate factorial using recursion and iteration |
| 6 | [Break & Continue Loop](./6.break_continue_loop) | Demonstrate break and continue statements in loops |
| 7 | [Sum of Odd Numbers (1 to N)](./7.one_to_n_odd_number_sum) | Calculate the sum of odd numbers from 1 to N |
| 8 | [Multiplication Table](./8.multiplicaion_table) | Generate multiplication table for a given number |
| 9 | [Prime Number Check](./9.prime_number) | Check if a given number is prime |
| 10 | [Array Input and Sum](./10.array_input_and_sum) | Take array input from user and calculate the sum of all elements |
| 11 | [Array Largest & Smallest Value](./11.array_largest_smallest_value_print) | Find and print the largest and smallest values in an array |
| 12 | [Reverse Array](./12.reverse_array) | Reverse an array using two-pointer technique |


## How to Run

Navigate to any problem directory and run:

```bash
cd 1.even_odd_number_check
go run main.go
```
